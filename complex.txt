1) Функция "complex" - создание комплексного числа:
	1 параметр - действительная часть,
	2 параметр - мнимая часть.
	return - complex

2) Функция "conj" - комплексно сопряженное число:
	1 параметр - комплексное число.
	return - complex

3) Функция "exp" - экспонента комплексного числа:
	1 параметр - комплексное число.
	return - complex

4) Функция "log" - логарифм комплексного число:
	1 параметр - комплексное число.	
	return - complex

5) Функция "pow" - степень комплексного числа:
	1 параметр - число (типы: complex или long double),
	2 параметр - в какую степень (типы: complex или long double).
	return - complex

6) Функция "sqrt" - квадратный корень комплексного числа:
	1 параметр - комплексное число.
	return - complex

7) Функция "abs" - модуль комплексного числа:
	1 параметр - комплексное число.
	return - long double

8) Функция "arg" - аргумент комплексного числа:
	1 параметр - комплексное число.
	return - long double

Для 1-8 возможно создания объекта из возвращаемого объекта.

Определены операции:
1) Сумма/разность/умножение/деление комплексных чисел.
2) Сумма/разность/умножение/деление комплексного числа и вещественного.

Примеры:
1) i = complex(0,1)
	создается переменная 'i': 0 + i

2) iConj = conj(i)
	создается переменная 'iConj': 0 - i

3) abs(i)
	выводится на экран значение модуля переменной i

4) sum = i + iConj
	создается переменная 'sum': 0 + 0i
