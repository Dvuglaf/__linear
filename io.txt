1) Имя переменной должно состоять только из прописных или заглавных символов латинского алфавита.


2) Переменная создается путем ввода имени переменной, оператора '=' ,вызов функции или ввод выражения(для вещественных чисел).
	variable = 2+2*2
	anotherVariable = vector(1 2 3 4 5)

3) Над ранее созданными переменными можно совершать разрешенные операции
	c = a+b 	#a и b должны быть созданы и должна быть поддержка оператора '+' для этих типов

4) Можно выводить значение выражения без сохранения в переменную, результат выведется на экран.
	3.14+99*3	#>> 3.14+99*3
			#
			# 300.14

5) Запрещен ввод двух и более функций в одной строке
	c = complex(1,1)
	c+complex(2,2)	# Ошибка!

6) Запрещено создание двух и более переменных в одной строке
	v = vector(1 2), c = complex(0,1)
			# Ошибка!

7) Вещественное число не обязательно сохранять в переменную
	c = complex(0,1)
	c*100		# Разрешено

8) Если существует переменная с каким-то именем, то при создании переменной с таким же именем, её значение перезапишется
	c = complex(1,1)
	c = vector(1 2 3)
			# c: (1, 2, 3)
