1) Функция "vector" - создание вектора:
	1 параметр - значения вектора, разделенные между собой пробелом.
	return - vector

2) Функция "vector" - создание вектора:
	1 параметр - начальное значение,
	2 параметр - конечное значение (не включительно),
	3 параметр - шаг значения (по умолчанию = 1).
	return - vector

3) Функция "cross" - векторное произведение (2-х или 3-х мерных векторов):
	1 параметр - вектор
	2 параметр - вектор
	return - vector

4) Функция "dot" - скалярное произведение
	1 параметр - вектор
	2 параметр - вектор	
	return - long double

Для 1-4 возможно создания объекта из возвращаемого объекта.

Определены операции:
1) Сумма/разность двух векторов.
2) Умножение вектора на вещественное число.

Примеры:
1) v = vector(1 2.9 3 4 5)
	создается вектор 'v':(1, 2.9, 3, 4, 5)

2) vv = vector(0.7, 1, 0.1)
	создается вектор 'v':(0.7, 0.8, 0.9)

3) firstVector = vector(1 2)
   secondVector = vector(0.7, 2, 0.9)
   cross(firstVector, secondVector)
